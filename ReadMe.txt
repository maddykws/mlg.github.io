The design submitted was converted into a webpage using HTML5 & CSS. 
Here are some considerations to generate the project: 
1. No Javascript was used. 
2. Div and table tags were used.
3. 80% of the CSS used contained % factors to enable Responsive Web Design.
4. External CSS is in place.
5. Z-indexes were used to create the overlapping layers.
6. CSS class style attributes were used.
7. At several places inline CSS was also used.
